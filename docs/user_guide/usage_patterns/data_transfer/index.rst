Data Transfer
=============

Globus provides numerous features and components aimed at robust and reliable
data transfer, integrating with your existing storage systems.

These docs focus on Globus Connect Personal, Globus Connect Server, and Globus
Transfer.
Additionally, integrating these components with other Globus services like
Timers is covered.
